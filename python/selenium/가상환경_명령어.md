* 아나콘다 가상환경 목록
```
$ conda info --evns
```

* 아나콘다 가상환경 생성
```
$ conda create --name [가상환경이름]
```

* 아나콘다 가상환경 삭제
```
$ conda remove --name [가상환경이름] --all
```

* 가상환경 실행/종료
```
$ conda activate [가상환경이름]
$ conda deactivate 
```
